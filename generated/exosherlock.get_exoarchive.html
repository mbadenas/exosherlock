exosherlock.get_exoarchive¶

exosherlock.get_exoarchive(table='exoplanets', fname: Union[str, pathlib.Path, IO[~AnyStr], NoneType] = None, default_pars: bool = False, local: bool = False, **kwargs) → pandas.core.frame.DataFrame[source]¶

Read a table form the NASA Exoplanet Archive as a pandas DataFrame.

Parameters
tablestr, optional

Which table from the exoplanet archive will be read. Used for custom dtypes which reduce memory usage and allow extra functionalities compared to automatic dtype parsing.

fnamestr, optional

Name of the file to import as dataframe. By default, the csv file provided by the package is loaded. Ignored if local==False.

default_parsbool, optional

If True, returns only the subset with default_flag == 1.

localbool, optional

When true, the local csv file at fname is read.

kwargs

Passed to pandas.read_csv()

Returns
pandas.DataFrame
